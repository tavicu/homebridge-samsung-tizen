# Introduction

Homebridge Samsung Tizen is a plugin created for Homebridge (opens new window) that allows you to control your Samsung TVs (models starting from 2017) that are running Tizen Operating System.

Please make sure to read our shiny documentation where you find step by step instructions with images on how to install and configure the plugin.

If you do have problems please make sure to check the common issues page and in case you want to open a new issue make sure to read details on how to open a new issue so we will have all the details needed to help you!

If you find this plugin useful and want to show your support then please star this plugin on GitHub, or better yet; you can donate through PayPal (opens new window) whatever you think the plugin value is. You can also use other methods by accessing the Donate menu from the top navbar.
